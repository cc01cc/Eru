## 1. 安装并配置 Tex Studio

1. 运行 texstudio 安装程序
   1. Latex Studio(可能被屏蔽) <https://www.texstudio.org/>
   2. Latex Studio(应该是个镜像，目前可以访问) <https://texstudio.sourceforge.net/>
2. 在 Tex Studio 的设置>构建选项卡中，调整默认编译器为XeLaTeX（XeLaTeX 采用 UTF-8 编码支持中日韩文字）

## 2. 常用快捷键

|快捷键|说明|
|--|--|
|F5|构建并查看|
|Ctrl+K|删除行|
|Alt+K|删除至行末|
